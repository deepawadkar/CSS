CSS BOX MODEL..!

Every element that can be displayed on a web page is comrised of one or more rectangular boxes. css box model typically describes how these rectangular boxes are laid out on a web page . These boxes can have diffrent properties and can interact with each other in diffrent ways, but every box has a content area and optional surrounding padding, border, and margin areas.
